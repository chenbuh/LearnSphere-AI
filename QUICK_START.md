# 🚀 快速开始指南

## 💻 启动方法

### Windows 用户
1. **双击运行** `启动软件.bat`
2. 等待自动打开浏览器
3. 开始学习！

### Mac/Linux 用户
1. **打开终端**，进入软件目录
2. **运行命令**：`./start.sh`
3. 等待自动打开浏览器
4. 开始学习！

### 手动启动（备选方案）
如果自动启动失败，可以：
1. 直接用浏览器打开 `src/html/index.html`
2. 或安装 Node.js 后运行 `node tools/launcher.js`

## 📋 系统要求

### 必需
- **浏览器**: Chrome 80+, Firefox 75+, Edge 80+, Safari 13+
- **系统**: Windows 7+, macOS 10.12+, Linux

### 可选（用于启动服务器）
- **Node.js** 12+ (推荐)
- 或 **Python** 3.6+

## 🎯 第一次使用

1. **选择考试类型**：四六级、雅思、托福等
2. **完成水平测试**：5-10分钟简单测试
3. **开始学习**：根据个人情况制定学习计划

## 🆘 常见问题

### Q: 启动失败怎么办？
A: 
1. 确保浏览器版本支持
2. 尝试重新下载软件
3. 直接打开 `src/html/index.html`

### Q: 数据会丢失吗？
A: 不会，所有学习数据保存在本地浏览器中

### Q: 需要网络连接吗？
A: 基础功能支持离线使用
