# 词汇完善工具使用说明

## 概述

本工具集用于完善英语学习系统中的词汇数据，包括音标、例句和例句翻译。使用免费的 Free Dictionary API 获取真实数据，如果API没有数据则自动生成高质量内容。

## 工具列表

### 1. 检查工具

#### `check_vocab_status.js` - 检查词汇状态
```bash
node check_vocab_status.js
```
显示数据库中词汇的整体状态，包括缺失音标、例句的统计。

#### `check_bad_examples.js` - 检查问题例句
```bash
node check_bad_examples.js
```
统计有问题的例句数量（如 "Example unavailable"、"This is useful" 等）。

### 2. 修复工具

#### `fix_all_vocabulary.js` - 修复所有缺失音标
```bash
node fix_all_vocabulary.js
```
- 自动修复所有缺失或错误的音标
- 使用 Free Dictionary API 获取标准音标
- 请求间隔 1.5 秒，避免API限流
- 如遇限流会自动等待60秒后重试

#### `fix_all_examples.js` - 修复所有问题例句
```bash
node fix_all_examples.js
```
- 修复所有问题例句（unavailable、This is useful等）
- 优先从 API 获取真实例句
- API无数据时自动生成高质量例句
- 同时生成中文翻译
- 每100个显示一次进度

### 3. 综合工具

#### `vocabulary_enhancer.js` - 词汇增强器
```bash
# 处理200个词汇
node vocabulary_enhancer.js --limit=200

# 处理指定考试类型
node vocabulary_enhancer.js cet4 --limit=100

# 同时更新JS文件
node vocabulary_enhancer.js --limit=100 --update-files
```
功能：
- 同时完善音标和例句
- 支持按考试类型处理
- 可选更新前端JS词汇文件

### 4. 导入工具

#### `import_vocabulary_to_db.js` - 从JS文件导入词汇
```bash
# 导入所有词汇文件
node import_vocabulary_to_db.js

# 导入指定考试类型
node import_vocabulary_to_db.js cet4
```
将前端 JS 词汇文件导入到数据库。

### 5. 批处理脚本

#### `run_vocabulary_enhancement.bat` - 一键运行工具
```bash
run_vocabulary_enhancement.bat
```
提供交互式菜单：
1. 导入词汇到数据库
2. 完善词汇（使用API）
3. 完善指定考试类型词汇
4. 查看数据库词汇统计
5. 退出

## API说明

### Free Dictionary API
- 地址: `https://api.dictionaryapi.dev/api/v2/entries/en/{word}`
- 免费，无需API Key
- 提供音标、定义、例句
- 限流：建议请求间隔 1.5-2 秒

## 支持的考试类型

- `cet4` - 大学英语四级
- `cet6` - 大学英语六级
- `toefl` - 托福
- `ielts` - 雅思
- `gre` - GRE
- `tem4` - 英语专业四级
- `tem8` - 英语专业八级
- `postgraduate` - 考研英语
- `primary` - 小学英语
- `middle_school` - 中考英语
- `high_school` - 高中英语
- `coca` - COCA语料库

## 使用流程

### 完整流程（首次使用）

1. **检查当前状态**
```bash
node check_vocab_status.js
node check_bad_examples.js
```

2. **修复音标**
```bash
node fix_all_vocabulary.js
```

3. **修复例句**
```bash
node fix_all_examples.js
```

4. **验证结果**
```bash
node check_vocab_status.js
```

### 日常维护

使用批处理脚本进行日常维护：
```bash
run_vocabulary_enhancement.bat
```

## 注意事项

1. **API限流**
   - 如遇 429 错误，脚本会自动等待60秒
   - 建议在网络稳定时运行
   - 大批量处理需要较长时间

2. **数据备份**
   - 运行前建议备份数据库
   - 可使用 mysqldump 备份

3. **进度监控**
   - 脚本会实时显示处理进度
   - 每批次显示统计信息
   - 可随时 Ctrl+C 中断

4. **错误处理**
   - 脚本会自动跳过失败的词汇
   - 最后显示失败统计
   - 可重新运行处理失败项

## 性能参数

- **请求间隔**: 1.5秒（可调整 DELAY 变量）
- **批次大小**: 100个（可调整 BATCH_SIZE 变量）
- **超时时间**: 10秒（API请求超时）
- **限流等待**: 60秒（遇到429错误）

## 故障排除

### 数据库连接失败
- 检查 MySQL 服务是否运行
- 验证数据库密码是否正确
- 确认数据库 `learnsphere_ai` 存在

### API请求失败
- 检查网络连接
- 确认可以访问 api.dictionaryapi.dev
- 增加请求间隔时间

### 内存不足
- 减小 BATCH_SIZE
- 分批处理不同考试类型
- 使用 --limit 参数限制处理数量

## 技术支持

如有问题，请检查：
1. Node.js 版本 >= 14
2. 已安装依赖: `npm install`
3. MySQL 版本 >= 5.7
4. 网络连接正常
