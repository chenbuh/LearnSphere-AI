## 🎯 文档使用指南

### 新手入门

如果你是第一次接触本项目，建议按以下顺序阅读文档：

1. **[项目README](../README.md)** - 了解项目概况
2. **[开发文档](./DEVELOPMENT.md)** - 搭建开发环境
3. **[后端文档](./BACKEND.md)** - 了解后端架构
4. **[API文档](./API.md)** - 熟悉接口调用

### 开发人员

如果你是开发人员，需要参与项目开发：

- **前端开发** → [开发文档](./DEVELOPMENT.md) + [API文档](./API.md)
- **后端开发** → [后端文档](./BACKEND.md) + [API文档](./API.md)
- **全栈开发** → 阅读所有文档

### 运维人员

如果你负责项目部署和运维：

- **部署上线** → [部署文档](./DEPLOYMENT.md)
- **问题排查** → [开发文档](./DEVELOPMENT.md) + [后端文档](./BACKEND.md)
