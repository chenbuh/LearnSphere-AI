# 词汇掌握度系统集成文档

## 1. 功能概述

词汇掌握度系统用于记录用户对单词的学习情况，并基于复习节奏给出下一次复习时间。

核心能力：

1. 掌握等级：按答题结果动态更新 0-5 级。
2. 复习计划：根据复习次数自动计算下一次复习时间。
3. 学习统计：区分已掌握、学习中、待开始。
4. 收藏与笔记：支持用户为单词做个性化标记。
5. 云端持久化：学习记录统一存储在后端数据库。

---

## 2. 数据库设计

### `vocabulary_mastery`

建议字段：

- `id`：主键
- `user_id`：用户 ID
- `vocabulary_id`：词汇 ID
- `mastery_level`：掌握等级（0-5）
- `review_count`：复习次数
- `correct_count`：答对次数
- `wrong_count`：答错次数
- `last_review_time`：上次复习时间
- `next_review_time`：下次复习时间
- `is_favorite`：是否收藏
- `notes`：用户笔记

### 复习间隔建议

| 复习次数 | 间隔（天） |
|---|---|
| 1 | 1 |
| 2 | 2 |
| 3 | 4 |
| 4 | 7 |
| 5 | 15 |
| 6+ | 30 |

答错时建议降低掌握度，并将复习时间提前。

## 3. 后端 API

### 3.1 记录学习结果

```http
POST /api/vocabulary/mastery/record
Content-Type: application/json

{
  "vocabularyId": 123,
  "isCorrect": true
}
```

### 3.2 获取复习列表

```http
GET /api/vocabulary/mastery/review-list?limit=20
```

### 3.3 获取统计数据

```http
GET /api/vocabulary/mastery/stats
```

### 3.4 收藏/取消收藏

```http
POST /api/vocabulary/mastery/favorite
Content-Type: application/json

{
  "vocabularyId": 123,
  "favorite": true
}
```

### 3.5 保存笔记

```http
POST /api/vocabulary/mastery/notes
Content-Type: application/json

{
  "vocabularyId": 123,
  "notes": "这个词常见于阅读题科技语境。"
}
```

## 4. 前端接入示例

示例 API 封装（`frontend-vue/src/api/mastery.js`）：

```js
import request from '@/utils/request'

export const masteryApi = {
  recordReview(vocabularyId, isCorrect) {
    return request.post('/vocabulary/mastery/record', { vocabularyId, isCorrect })
  },
  getReviewList(limit = 20) {
    return request.get('/vocabulary/mastery/review-list', { params: { limit } })
  },
  getStats() {
    return request.get('/vocabulary/mastery/stats')
  },
  toggleFavorite(vocabularyId, favorite) {
    return request.post('/vocabulary/mastery/favorite', { vocabularyId, favorite })
  },
  addNotes(vocabularyId, notes) {
    return request.post('/vocabulary/mastery/notes', { vocabularyId, notes })
  }
}
```

## 5. 掌握等级建议

- `0`：未学习
- `1-2`：学习中
- `3-4`：较熟悉
- `5`：已掌握

可根据正确率与复习次数组合判定等级。

## 6. 验证与排查

### 6.1 快速验证

```bash
curl -X POST http://localhost:8080/api/vocabulary/mastery/record \
  -H "Content-Type: application/json" \
  -H "satoken: YOUR_TOKEN" \
  -d '{"vocabularyId":1,"isCorrect":true}'

curl http://localhost:8080/api/vocabulary/mastery/stats \
  -H "satoken: YOUR_TOKEN"
```

### 6.2 常见问题

1. 统计不更新：检查后端日志与数据库写入。
2. 复习列表为空：确认用户是否已有学习记录。
3. 前端调用失败：检查 token、网关路径、跨域配置。
