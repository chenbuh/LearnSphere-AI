# 复习页面和统计面板 - 路由配置

## 路由配置

在 `frontend-vue/src/router/index.js` 中添加：

```javascript
{
  path: '/review',
  name: 'Review',
  component: () => import('../views/ReviewView.vue'),
  meta: { requiresAuth: true }
}
```

## 在导航菜单中添加入口

### 1. 在顶部导航添加（main layout）

```vue
<template>
  <n-menu>
    <n-menu-item key="vocabulary" @click="router.push('/vocabulary')">
      词汇学习
    </n-menu-item>
    <n-menu-item key="review" @click="router.push('/review')">
      智能复习
    </n-menu-item>
    <!-- 其他菜单项 -->
  </n-menu>
</template>
```

### 2. 在词汇页面添加快捷入口

在 `VocabularyView.vue` 的统计卡片区域添加：

```vue
<n-grid-item>
  <n-card class="stat-card review" :bordered="false" clickable @click="router.push('/review')">
    <div class="stat-content">
      <div class="stat-icon review">
        <RotateCw />
      </div>
      <div class="stat-info">
        <div class="stat-label">待复习</div>
        <div class="stat-value">{{ reviewCount }}</div>
      </div>
    </div>
  </n-card>
</n-grid-item>
```

### 3. 在个人中心显示统计

在 `ProfileView.vue` 中引入 StatsPanel：

```vue
<script setup>
import StatsPanel from '@/components/StatsPanel.vue'
</script>

<template>
  <div class="profile-view">
    <!-- 用户信息 -->
    <UserInfo />
    
    <!-- 学习统计 -->
    <StatsPanel />
  </div>
</template>
```

## 完整示例

### router/index.js

```javascript
import { createRouter, createWebHistory } from 'vue-router'

const routes = [
  {
    path: '/',
    name: 'Home',
    component: () => import('../views/HomeView.vue')
  },
  {
    path: '/vocabulary',
    name: 'Vocabulary',
    component: () => import('../views/VocabularyView.vue'),
    meta: { requiresAuth: true }
  },
  {
    path: '/review',
    name: 'Review',
    component: () => import('../views/ReviewView.vue'),
    meta: { requiresAuth: true }
  },
  // ... 其他路由
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```

## 使用方式

### 访问复习页面

```
http://localhost:5173/review
```

### 在代码中跳转

```javascript
import { useRouter } from 'vue-router'

const router = useRouter()

// 跳转到复习页面
router.push('/review')
```

- ✅ 圆形进度条（学习进度）
- ✅ 4个详细统计卡片
- ✅ 实时计算占比
- ✅ 个性化学习建议
- ✅ 响应式设计
- ✅ 悬停动画效果
