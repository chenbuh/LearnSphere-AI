## 🚀 使用方法

### 1. 启动后端
确保Spring Boot应用正在运行

### 2. 访问每日任务页面
```
http://localhost:5173/daily-tasks
```

### 3. API测试

#### 创建学习计划
```bash
curl -X POST http://localhost:8080/api/study-plan/create \
  -H "Content-Type: application/json" \
  -H "satoken: YOUR_TOKEN" \
  -d '{
    "examType": "cet4",
    "targetScore": 550,
    "durationDays": 30
  }'
```

#### 获取今日任务
```bash
curl http://localhost:8080/api/study-plan/tasks/today \
  -H "satoken: YOUR_TOKEN"
```

#### 完成任务
```bash
curl -X POST http://localhost:8080/api/study-plan/tasks/1/complete \
  -H "Content-Type: application/json" \
  -H "satoken: YOUR_TOKEN" \
  -d '{"completedCount": 50}'
```
