# 错题本与口语练习功能更新

## 1. 错题本升级 (Error Book)

### ✨ 新增功能
- **分类浏览**：新增了标签页 (Tabs)，现在可以按“全部”、“词汇”、“语法”、“阅读”、“听力”分类查看错题。
- **分页加载**：实现了服务器端分页，避免一次性加载过多数据，提高性能。
- **真实数据**：移除了模拟数据，现在直接显示用户的实际错题记录。
- **数据展示优化**：针对不同类型的题目（如阅读、词汇），智能提取并展示题目内容和解析。

### 🔧 技术实现
- **前端**：重构 `ErrorBookView.vue`，引入 `NTabs` 和 `NPagination` 组件。
- **后端**：
  - 修改 `ILearningRecordService` 和 `LearningRecordServiceImpl`，在 `getUserRecords` 方法中增加 `isCorrect` 参数。
  - 更新 `LearningRecordController` 的 `/api/learning/records` 接口，支持传递 `isCorrect=0` 来只查询错题。

## 2. 口语练习增强 (Speaking Practice)

### ✨ 新增功能
- **最近生成话题**：在口语练习的开始页面（Setup 阶段）新增了“最近生成话题”板块。
- **快速回顾**：用户可以直接点击历史记录中的话题，快速重新加载并开始练习。
- **分页浏览**：支持查看更早的历史记录。

### 🔧 技术实现
- **前端**：在 `SpeakingView.vue` 的 Setup 视图区域复用了历史记录组件，绑定了现有的分页状态和加载逻辑。

## 🚀 部署指南

1. **后端**：需要重新编译后端代码，因为修改了 Controller 和 Service 层的 Java 代码。
   ```bash
   cd backend
   mvn clean package -DskipTests
   # 重启后端服务
   ```

2. **前端**：无需特殊构建，保存文件即可（如果是热重载开发环境）。生产环境需重新构建。
