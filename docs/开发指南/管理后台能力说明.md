# LearnSphere AI 管理后台高级能力说明

本文档记录了近期为提升管理后台运维效率与高级感而引入的各项进阶功能。

## 1. 全局指挥中心 (Command Palette) - `Ctrl + K`

通过快捷键 `Ctrl + K` 可唤起全局搜索与指挥面板，瞬间提升操作效率。

### 核心能力：
- **瞬时导航**：输入页面名称（如“用户”、“日志”）即可跨页面跳转。
- **关联搜索**：支持搜索业务数据，包括：
    - **用户信息**：按用户名、邮箱实时检索。
    - **词汇库**：按单词、翻译实时检索。
    - **学习内容**：检索阅读文章、写作题目等。
- **快捷操作**：集成快速动作，如“快速添加用户”、“打开 AI 沙箱”等。
- **极简设计**：采用毛玻璃（Glassmorphism）特效，配备流畅的交互动画及数据类型标签。

## 2. Redis 缓存可视化管理

作为运维利器，支持实时管理系统缓存。

### 功能亮点：
- **实时探测**：支持按 Pattern 扫描 Redis 缓存键。
- **深度透视**：点击详情可直接查看缓存的 **JSON 数据结构**、**剩余过期时间 (TTL)** 及数据类型。
- **安全清理**：支持精确删除单个键，或按搜索匹配项批量清理（带安全确认逻辑）。
- **运维提效**：解决模型更新或配置变更后手动清理缓存的烦恼。

## 3. 全局运维监控 (System Monitor) - Updated

全新升级的系统状态监控面板，提供沉浸式的服务器资源可视化体验。

### 功能亮点：
- **现代化 UI**：采用渐变色卡片与大图标设计，关键指标一目了然。
- **实时数据流**：秒级刷新 CPU、负载、内存、磁盘等核心指标。
- **可视化仪表盘**：
    - **JVM 堆内存**：动态进度条展示不同内存区域使用情况。
    - **磁盘空间**：环形进度图直观显示存储占用。
- **环境透视**：底部胶囊式布局清晰展示 Java 版本、JVM 参数及启动时间。

## 4. 技术背景
- **前端**：Vue 3 + Naive UI + Lucide Icons + TailwindCSS。
- **后端**：Spring Boot + Spring Data Redis + OSHI (System Info)。
- **安全**：所有操作均通过 Sa-Token 鉴权，仅限拥有对应权限的管理员执行。
