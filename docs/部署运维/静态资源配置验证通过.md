// 带 contenthash 的静态资源（JS/CSS/字体/图片），设置长期缓存

## 📝 缓存策略

| 资源类型 | 路径 | 缓存时间 | 说明 |
|---------|------|---------|------|
| 带 hash 的资源 | `/assets/**` | 7 天 | 内容不变 URL 不变，可长期缓存 |
| index.html | `/`, `/dist/index.html` | 不缓存 | 每次都获取最新版本 |
| Service Worker | `/sw.js` | 不缓存 | 确保及时更新 |
| 管理后台 | `/admin/**` | 不缓存 | 每次都获取最新版本 |

## ⚠️ 注意事项

1. **前端修改后必须重新构建**
   ```bash
   npm run build:frontend
   cd backend && mvn clean package -DskipTests
   ```

2. **管理后台需要手动同步**
   ```bash
   npm run build:admin
   npm run sync:admin
   ```

3. **Service Worker 缓存问题**
   - 如果页面不更新，清除浏览器缓存
   - 或使用隐私模式测试

4. **移动端音频播放**
   - 必须用户交互后才能播放（点击按钮）
   - 不支持自动播放
