# LearnSphere AI 宝塔后端部署保姆级教程

本文档提供了将 LearnSphere AI 后端服务部署到 Linux 宝塔面板的完整流程，涵盖从环境准备到项目上线的每一个细节。

---

## 🛠️ 1. 环境准备

在宝塔面板的「软件商店」中，确保已安装并配置好以下软件：

1.  **JDK 17**: 必须使用 JDK 17（Spring Boot 3.x 要求）。
    *   在「Java项目管理器」或「容器脚本」中安装。
2.  **MySQL 8.0+**: 建议使用 8.0 版本，确保字符集为 `utf8mb4`。
3.  **Redis 6.0+**: 系统用于管理 VIP 配额和缓存，必须启动。
4.  **Python 3.10+**: 用于边缘 TTS 语音合成引擎（如果需要语音功能）。
    *   检查命令：在终端输入 `python3 --version`。
5.  **Nginx**: 用于反向代理接口。

---

## 💾 2. 数据库初始化

1.  **创建数据库**:
    *   数据库名: `learnsphere_ai`
    *   字符集: `utf8mb4`
2.  **导入脚本**:
    *   上传并执行项目根目录下的 `backend/learnsphere_ai_production.sql`。
    *   该脚本包含最新的系统设置和 4.9 万条基础词库数据。
3.  **（可选）手动创建用户**:
    *   确保数据库用户（如 `chen`）拥有对该库的全部权限。

---

## 📦 3. 后端打包与上传

1.  **本地打包**:
    在本地 `backend` 目录下执行：
    ```bash
    mvn clean package -DskipTests
    ```
2.  **上传文件**:
    在服务器创建目录 `/www/wwwroot/learnsphere-backend/`，上传以下文件：
    *   `target/learnsphere-ai-backend-1.0.0.jar`
    *   `requirements-voice.txt`（位于 backend 源码根目录）

---

## 🔐 4. 安全与配置 (外置配置文件)

为保护隐私，建议不要在 Jar 包内写死密码。

1.  **创建配置文件夹**: `/www/wwwroot/learnsphere-backend/config/`
2.  **编写配置文件**: `application-secret.properties`
    ```properties
    # 数据库连接
    spring.datasource.username=chen
    spring.datasource.password=chen20040209
    
    # Redis (如果有密码)
    # spring.redis.password=你的密码
    
    # AI 核心密钥 (百炼/OpenAI)
    ai.api-key=你的API密钥
    ```

---

## 🚀 5. 宝塔 Java 项目添加

进入「Java项目管理器」->「添加项目」：

*   **项目 Jar 路径**: `/www/wwwroot/learnsphere-backend/learnsphere-ai-backend-1.0.0.jar`
*   **项目端口**: `8080`
*   **JVM 参数** (放在 `-jar` 之前):
    ```text
    -Xms256M -Xmx1024M -Dfile.encoding=UTF-8 -DJASYPT_PASSWORD=你的解密密钥(如果有)
    ```
*   **程序参数** (放在 Jar 包之后):
    ```text
    --spring.profiles.active=prod --spring.config.additional-location=file:/www/wwwroot/learnsphere-backend/config/
    ```

> [!IMPORTANT]
> `--spring.config.additional-location` 必须以 **`/`** 结尾，否则 Spring 无法将其识别为目录。

---

## 🎙️ 6. 语音引擎依赖 (核心步骤)

在服务器终端执行以下命令，安装 Python 依赖：

```bash
cd /www/wwwroot/learnsphere-backend
pip3 install -r requirements-voice.txt
```

*   **注意**: 如果提示 `pip3` 未找到，请先安装：`apt update && apt install -y python3-pip`。
*   **适配**: 如果 `python3` 命令不可用，在程序参数中显式指定：`--voice-engine.python-command=python`。

---

## 🌐 7. Nginx 反向代理

在宝塔「网站」设置中，为你的域名添加反向代理（或直接修改配置）：

```nginx
location ^~ /api/ {
    proxy_pass http://127.0.0.1:8080/;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
}
```

---

## 🔍 8. 验收清单

1.  **启动状态**: 项目状态显示「运行中」。
2.  **健康检查**: 访问 `https://你的域名/api/health/check`。
    *   成功返回: `{"status":"UP"}`。
3.  **语音测试**: 在词汇详情页点击播放，检查后端日志是否有 `Edge TTS` 启动记录。
4.  **数据库连接**: 若报 `Access denied`，检查 `application-secret.properties` 路径和密码。

---

## ❓ 常见问题 (FAQ)

*   **502 错误**: 后端没启动。检查宝塔项目日志（Log）。
*   **配置没生效**: 检查 `--spring.config.additional-location` 路径是否正确。
*   **语音报错**: 检查 `pip install` 是否成功，及是否具备 Python3 环境。
