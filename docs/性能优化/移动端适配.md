# 移动端适配指南 (Mobile Adaptation Guide)

## 概述
LearnSphere AI（包含 `frontend-vue` 和 `admin-vue`）实现了全功能全局的移动端自动适配。采用流体系统与 CSS 变量方案，无需在单个 `.vue` 文件中手动处理大多数自适应问题。

## 适配方案

本项目的适配主要依赖以下全局样式文件：

1. **`global-mobile.css`** 
   - 包含特定视口宽度的响应式媒体查询 `@media (max-width: 768px)`
   - 自动调整 `.n-card`、`.n-data-table`、`.n-form-item` 等基础 Naive UI 组件的内边距、字体等
   - 强制将硬编码的宽度的弹窗（如 `width: 800px` 的 `n-modal`）约束在最大 `95vw` 以内，确保在小屏幕中不溢出。

2. **`fluid-adaptive.css`**
   - 定义基础字号与流体尺寸：如利用 `clamp` 函数自动处理文字与间距。
   - `clamp(min, preferred, max)` 的广泛应用保证界面随屏幕自然且平滑地缩放。

## 适用范围
- **前端用户界面 (frontend-vue):** 包含完整学习面板、词汇测试、背单词等组件响应式
- **管理后台界面 (admin-vue):** 控制台与操作台全面移动化，左侧导航栏在小屏自动转换为抽屉或侧滑菜单式。

## 开发建议
1. 使用流式类名（如 Flex, Grid），减少硬编码宽度（如 `width: 500px`），推荐使用 `max-width: 100%`。
2. 调用 Naive UI 组件时，尽量使用默认流式宽高。如必须固定，请确认该弹窗或容器在移动端的媒体查询中会被覆写。
3. 测试时可通过开发者工具切换出 "iPhone Pro" 或其他小屏设备，确认组件无横向滚动条出现。
