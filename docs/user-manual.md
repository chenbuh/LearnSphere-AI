# 📖 用户使用手册

## 🚀 软件安装与启动

### 💻 系统要求
- **操作系统**: Windows 7/8/10/11, macOS 10.12+, Linux (Ubuntu 16.04+)
- **浏览器**: Chrome 80+, Firefox 75+, Edge 80+, Safari 13+
- **内存**: 最低 512MB，推荐 1GB+
- **存储空间**: 最低 200MB，推荐 500MB+
- **网络**: 可选（支持离线使用）

### 📥 安装步骤

#### Windows 用户
1. **下载软件包** - 从官网下载 `英语等级考试学习软件.zip`
2. **解压文件** - 解压到任意目录（建议桌面或D盘）
3. **启动软件** - 双击 `启动软件.exe` 或 `start.bat`
4. **自动跳转** - 软件将在默认浏览器中自动打开

#### Mac/Linux 用户
1. **下载软件包** - 下载对应平台的压缩包
2. **解压文件** - 解压到合适的目录
3. **运行脚本** - 执行 `./start.sh` 启动脚本
4. **浏览器打开** - 软件将在默认浏览器中启动

### 🔧 首次设置

#### 1. 选择学习目标
- 进入软件后，首先选择您的学习目标：
  - 🎓 **大学英语四级 (CET-4)**
  - 🎓 **大学英语六级 (CET-6)**
  - 🎓 **考研英语一/二**
  - 🌍 **雅思考试 (IELTS)**
  - 🌍 **托福考试 (TOEFL)**
  - 📚 **其他考试类型**

#### 2. 水平测试
- 完成5-10分钟的快速水平测试
- 系统将根据测试结果制定个性化学习计划
- 可以随时重新测试调整学习难度

#### 3. 个人设置
- **学习时间**: 设置每日学习目标时间
- **提醒设置**: 配置学习提醒通知
- **界面主题**: 选择亮色或暗色主题
- **语音设置**: 配置语音播放和识别

## 📚 主要功能使用

### 🔤 词汇学习模块

#### 📖 基础词汇学习
1. **进入词汇模块**
   - 点击主界面的 "词汇学习" 按钮
   - 选择词汇分类（基础、进阶、专业等）

2. **学习新单词**
   - 系统自动推荐需要学习的单词
   - 显示单词的发音、释义、例句
   - 点击 🔊 图标听取标准发音

3. **记忆测试**
   - 完成单词选择题
   - 进行拼写练习
   - 语音识别发音练习

#### 🎮 词汇游戏
1. **单词消除游戏**
   - 连接相同意思的单词进行消除
   - 在游戏中加深单词记忆

2. **单词接龙**
   - 根据词根或首字母进行单词接龙
   - 挑战更长的单词链

3. **拼写蜜蜂**
   - 听音频拼写正确的单词
   - 限时挑战模式

#### 📊 学习统计
- **学习进度**: 查看已学习的单词数量和比例
- **掌握程度**: 显示每个单词的记忆强度
- **复习提醒**: 根据遗忘曲线安排复习

### 📝 语法练习模块

#### 📖 语法知识学习
1. **选择语法章节**
   - 从语法目录中选择要学习的章节
   - 系统推荐薄弱的语法点

2. **知识点讲解**
   - 详细的语法规则说明
   - 丰富的例句演示
   - 中英对比分析

3. **互动练习**
   - 即时的练习题测试
   - 错误解析和正确答案说明

#### 🎯 专项练习
1. **选择题练习**
   - 单选、多选语法题
   - 实时评分和解析

2. **填空题练习**
   - 语法填空练习
   - 词汇填空练习

3. **改错题练习**
   - 识别并纠正语法错误
   - 解释错误原因

#### 📈 进步追踪
- **正确率统计**: 各语法点的正确率
- **薄弱分析**: 识别需要加强的语法点
- **学习建议**: 个性化的学习建议

### 🎵 听力训练模块

#### 🎧 听力材料
1. **选择听力类型**
   - 日常对话、学术讲座、新闻报道等

2. **播放控制**
   - ⏯️ 播放/暂停

3. **字幕设置**
   - 中文字幕、英文字幕或无字幕

#### 🎯 训练模式
1. **精听模式**

   - 填空练习
   - 跟读练习

2. **泛听模式**
   - 整体理解训练

   - 态度观点理解

3. **听写模式**
   - 边听边写练习
   - 提高听力记录能力

#### 📊 听力分析
- **理解准确率**: 听力理解正确率统计
- **语速适应性**: 不同语速下的表现
- **题型分析**: 各类听力题型的得分情况

## 🆘 常见问题解答

### 🔧 技术问题

#### Q: 软件启动失败怎么办？
**A:** 请按以下步骤排查：
1. 确认浏览器版本符合要求
2. 检查防火墙是否阻止软件运行
3. 尝试更换浏览器启动
4. 重新下载并解压软件包

#### Q: 音频无法播放？
**A:** 解决方案：
1. 检查浏览器音频权限设置
2. 确认系统音量未静音
3. 更新浏览器到最新版本
4. 清除浏览器缓存后重试

#### Q: 数据丢失了怎么办？
**A:** 数据恢复方法：
1. 检查浏览器本地存储是否被清除
2. 查看自动备份文件
3. 联系技术支持恢复数据

### 📚 使用问题

#### Q: 如何更换学习目标？
**A:** 
1. 进入 "设置" 页面
2. 选择 "学习目标"
3. 重新选择考试类型
4. 完成新的水平测试

#### Q: 可以离线使用吗？
**A:** 
- 基础功能支持离线使用
- 需要先在线加载必要数据
- 统计数据会在联网时同步

#### Q: 如何重置学习进度？
**A:**
1. 进入 "设置" → "数据管理"
2. 选择 "重置学习进度"
3. 确认操作（此操作不可恢复）
