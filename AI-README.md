# 🤖 AI智能推荐系统

## 概述

本项目集成了完整的AI智能推荐系统，为英语学习者提供个性化的学习建议和薄弱点分析。系统采用模块化设计，支持实时分析、自适应学习路径和智能内容推荐。

## ✨ 核心功能

### 1. 智能薄弱点分析
- **多维度评估**：从准确率、时间花费、进步速度等多个维度分析学习表现
- **实时监控**：自动跟踪学习活动，实时更新薄弱点分析
- **遗忘曲线预测**：基于艾宾浩斯遗忘曲线预测知识点遗忘风险
- **知识图谱关联**：利用知识点之间的关联性发现隐藏的薄弱环节

### 2. 个性化推荐引擎
- **混合推荐算法**：结合协同过滤和内容过滤技术
- **上下文感知**：考虑学习时间、设备类型、可用时间等因素
- **置信度评分**：为每个推荐提供可信度评估
- **动态调整**：根据用户反馈和学习效果动态调整推荐策略

### 3. 自适应学习路径
- **个性化路径生成**：基于用户能力和学习目标生成最优学习路径
- **认知负载管理**：避免学习过载，确保最佳学习效果
- **多元智能理论**：考虑不同学习者的智能类型偏好
- **实时路径调整**：根据学习进度动态调整后续学习内容

### 4. 学习表现追踪
- **全方位数据收集**：记录所有学习活动和表现数据
- **智能统计分析**：生成详细的学习报告和趋势分析
- **行为模式识别**：识别学习习惯和行为模式
- **预测性分析**：预测学习效果和潜在问题

## 🏗️ 系统架构

### 模块组成
```
AI推荐系统/
├── WeaknessAnalyzer (薄弱点分析器)
├── RecommendationEngine (推荐引擎)
├── AdaptiveLearningPath (自适应学习路径)
├── PerformanceTracker (表现追踪器)
└── AIRecommendationManager (AI管理器)
```

### 数据流
1. **数据收集** → PerformanceTracker收集学习活动
2. **薄弱点分析** → WeaknessAnalyzer分析用户薄弱点
3. **推荐生成** → RecommendationEngine生成个性化推荐
4. **路径规划** → AdaptiveLearningPath制定学习路径
5. **反馈优化** → 根据用户反馈优化推荐算法

## 🚀 快速开始

### 方式一：使用演示页面（推荐）
1. 双击运行 `启动AI演示.bat`

4. 点击"生成模拟数据"创建学习记录
5. 体验各种AI功能

### 方式二：集成到主应用
1. 打开 `src/html/index.html`
2. AI推荐系统会自动初始化
3. 在首页查看推荐内容和薄弱点分析

## 🔧 技术实现

### 核心算法
1. **薄弱点检测算法**
   - 基于统计分析的多维度评估
   - 考虑时间衰减和学习频次
   - 知识点关联性分析

2. **推荐算法**
   - 协同过滤：基于相似用户的推荐
   - 内容过滤：基于内容特征的推荐
   - 混合推荐：结合多种算法的综合推荐

3. **学习路径优化**
   - 遗传算法：优化学习序列
   - 强化学习：根据反馈调整策略
   - 约束优化：考虑时间和认知负载限制

### 数据存储
- **主存储**：IndexedDB（浏览器本地数据库）
- **缓存机制**：内存缓存提升访问速度
- **降级支持**：LocalStorage和内存存储作为备选

### 性能优化
- **异步处理**：所有AI计算采用异步方式
- **缓存策略**：智能缓存减少重复计算
- **批量处理**：批量处理学习活动数据
- **懒加载**：按需加载AI模块

## 🎯 使用建议

### 最佳实践
1. **数据积累**：建议至少有50+学习记录后再使用AI推荐
2. **定期更新**：建议每周重新生成推荐内容
3. **反馈循环**：根据推荐效果调整学习策略
4. **多样化学习**：结合AI推荐和自主学习

### 注意事项
- AI系统需要一定时间的数据积累才能提供准确推荐
- 推荐结果仅供参考，应结合个人实际情况使用
- 定期清理过期的缓存数据以保持系统性能
