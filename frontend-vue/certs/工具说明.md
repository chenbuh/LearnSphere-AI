## 📁 目录结构

### `certs/` - HTTPS 开发证书
**用途**: 配置 HTTPS 以支持移动端录音功能

#### 文件说明
- `README.md` - HTTPS 配置说明文档
- `generate-cert.bat` - Windows 证书生成脚本（可选，推荐使用 mkcert）

#### 使用方法
```bash
# 1. 安装 mkcert（推荐）
choco install mkcert
mkcert -install

# 2. 启动开发服务器（自动使用 HTTPS）
cd ..
npm run dev

# 3. 访问 https://localhost:5173
```

详细说明请查看：`certs/README.md`

### 音标修复工具
**用途**: 修复 TEM-4、TEM-8、COCA 单词数据的音标问题

#### 1. `fix-tem4-phonetics.js`
**用途**: 将 TEM-4 的旧式音标转换为标准 IPA

**运行方式**:
```bash
cd src/data
node fix-tem4-phonetics.js
```

**效果**:
- 修复前: `/AkE demik/`
- 修复后: `/ækædemɪk/`

#### 2. `fetch-phonetics.py`
**用途**: 批量从在线词典获取单词音标

**运行方式**:
```bash
# 安装依赖
pip install requests

# 处理前 100 个单词（测试）
python fetch-phonetics.py tem8_words.js 0 100

# 处理全部单词
python fetch-phonetics.py tem8_words.js
```

#### 3. `phonetic-dict.js`
**用途**: 内置的常用词音标字典（约 500 个）

**使用方式**:
```javascript
import { phoneticDict } from './phonetic-dict.js'

const phonetic = phoneticDict['abandon'] // "/əˈbændən/"
```
