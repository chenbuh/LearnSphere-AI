# 🎓 英语等级考试学习软件

> 一个有趣、互动且完全免费的英语等级考试学习平台

## 📋 项目简介

这是一个基于Web技术开发的英语等级考试学习软件，专为想要提高英语水平并准备各类英语等级考试的学习者设计。软件通过点击启动后在浏览器中运行，提供丰富的学习功能和有趣的互动体验。

### ✨ 主要特色

- 🆓 **完全免费** - 无任何付费功能，所有内容完全开放
- 🎮 **游戏化学习** - 通过游戏化元素让学习更有趣
- 🌐 **浏览器运行** - 点击启动后在浏览器中运行，无需复杂安装
- 📱 **响应式设计** - 支持电脑、平板、手机等多种设备
- 🎯 **多种考试类型** - 支持四六级、雅思、托福、考研英语等
- 📊 **智能统计** - 详细的学习进度和成绩分析
- 🎵 **语音支持** - 听力练习和发音纠正功能

## 🎯 支持的考试类型

### 🇨🇳 国内考试
- **大学英语四级 (CET-4)**
- **大学英语六级 (CET-6)**
- **考研英语一/二**
- **专业英语四级/八级 (TEM-4/8)**
- **公共英语等级考试 (PETS)**

### 🌍 国际考试
- **雅思考试 (IELTS)**
- **托福考试 (TOEFL)**
- **GRE考试**
- **GMAT考试**

## 🚀 快速开始

### 启动方法
1. 双击 `启动软件.exe` 或 `start.bat`
2. 软件将自动在默认浏览器中打开
3. 选择您的目标考试类型
4. 开始个性化学习之旅

### 系统要求
- Windows 7 或更高版本
- 现代浏览器（Chrome、Firefox、Edge、Safari）
- 至少512MB可用内存

## 📚 学习模式

### 📖 基础学习模式
- **词汇学习** - 分级词汇，智能记忆曲线
- **语法练习** - 交互式语法讲解和练习
- **阅读理解** - 分级阅读材料，逐步提高
- **听力训练** - 真实语境听力材料

### 🎮 游戏化模式
- **单词大作战** - 在游戏中记忆单词
- **语法冒险** - 通过闯关学习语法
- **听力挑战** - 限时听力游戏
- **阅读竞赛** - 与AI或其他用户比拼

### 📝 模拟考试模式
- **真题模拟** - 历年真题完整模拟
- **专项训练** - 针对薄弱环节强化
- **限时练习** - 提高答题速度
- **错题回顾** - 智能错题分析
