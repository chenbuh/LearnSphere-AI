# 批量更新例句说明

## 📋 概述

这些脚本用于替换数据库中重复的例句，将通用的例句替换为真实、有意义的例句。

## 📊 当前状态

- **总词汇数**: 18,162 条

## 🚀 使用方法

### 方式1：一次性运行所有批次
```bash
run_all_batches.bat
```
这会自动执行批次1-10，总共更新100条记录。

### 方式2：单独运行每个批次
```bash
run_batch1.bat   # 批次1: 10条记录
run_batch2.bat   # 批次2: 10条记录
run_batch3.bat   # 批次3: 10条记录
...
run_batch10.bat  # 批次10: 10条记录
```

### 验证更新结果
```bash
verify_updates.bat
```
